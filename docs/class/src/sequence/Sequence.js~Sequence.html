    Sequence

    Sequence: a string of characters representing a sequence of nucleotides Usage: let sequence = new Sequence('ACCTGG', DNAAlphabet); let transcribed = sequence.transcribe(); transcribed.sequence; => 'ACCUGG' transcribed.getAlphabet(); => 'ACGU' // now an RNA strand
